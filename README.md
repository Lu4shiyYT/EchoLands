# Echo Lands

A 2D side-scrolling pixel-art sandbox survival game inspired by Terraria.
Built with pure HTML5, CSS3, and vanilla JavaScript — no libraries, no frameworks.

## How to Run

### Option A: Local
1. Clone or download this repository.
2. Open `index.html` directly in any modern browser (Chrome/Firefox recommended).
   - Note: Some browsers block ES features on `file://`. If issues occur, use a local server.

### Option B: Local Server (recommended)
```bash
# Using Python 3
python -m http.server 8080
# Then open http://localhost:8080
